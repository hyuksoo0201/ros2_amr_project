# hs_topview

Top-view camera 기반 주행 제어 패키지입니다.
이 파일은 패키지 배포/참조용 포인터 문서입니다.

실행 방법, 정책, 트러블슈팅, 인터페이스의 기준 문서는 루트 README입니다.

- 기준 문서: [../../README.md](../../README.md)
- Launch 파일: `launch/point_move.launch.py`, `launch/waypoint_sender.launch.py`
- 파라미터 파일: `params/point_move.yaml`, `params/waypoints.yaml`
- 노드 구현: `hs_topview/point_move.py`, `hs_topview/waypoint_sender.py`
